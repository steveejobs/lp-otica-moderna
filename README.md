# Ótica Moderna Araguaína

Landing page premium para a Ótica Moderna Araguaína.

## Deploy no Vercel

Configuração esperada:
- Framework Preset: `Next.js`
- Build Command: `npm run build`
- Install Command: `npm install`
- Output Directory: deixe vazio

Rotas principais:
- `/`: site completo com hero, consultoria, galerias separadas, Ray-Ban, vídeos, dicas, depoimentos e localização.
- `/instagram`: bio compacta premium para o Instagram.
- `/bio`: variação compacta para links.
- `/proposta`: página de proposta.

Assets oficiais normalizados em `public/assets/otica-moderna`.
